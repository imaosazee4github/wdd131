LDSConnect – Site Plan

A planning document for the LDSConnect project

Site Name

LDSConnect – A Church News and Events Website. This name was chosen to represent a platform where members can stay informed and connected with church updates, events, and spiritual messages.

Optional Domain: ldsconnect.org

Site Purpose

The site will feature recent church news, upcoming events, and inspirational articles. Members can also interact by browsing events and submitting questions via a contact form.

Scenarios

  • What upcoming church events are scheduled near my area?
  • Where can I find the latest announcements or inspirational messages?
  • How do I contact the church leaders for questions or feedback?

Color Schema

The selected colors are:

  • Primary Color: #2C3E50 – used for headers and navigation background.
  • Secondary Color: #E67E22 – used for accents, buttons, and highlights.
  • Background Color: #F9F9F9 – used for the main background.

Typography

Fonts chosen:

  • Montserrat – used for headings.
  • Open Sans – used for body text and paragraphs.
